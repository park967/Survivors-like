# Night Bloom Survivors 공략집

브라우저에서 바로 실행하는 작은 뱀서라이크 프로토타입입니다.  
플레이어는 이동만 하고, 공격은 자동으로 나갑니다.

## 실행 방법

`index.html`을 더블클릭해서 실행합니다.

```text
C:\Users\박범수\Documents\lol\index.html
```

개발용 로컬 서버로 보고 싶으면 아래 명령을 사용할 수 있습니다.

```powershell
node server.mjs
```

## 임시 이미지 자산

현재 캐릭터, 기본 적, 브루트 적, 경험치 보석은 임시 스프라이트 시트 이미지를 사용합니다.

```text
assets/sprites.png
```

원본 생성 이미지는 아래에 보관되어 있습니다.

```text
assets/sprites-source.png
```

나중에 직접 그린 이미지로 바꾸려면 같은 방식의 2x2 스프라이트 시트로 교체하거나, `game.js`의 `spriteCells`와 `drawSprite()` 연결을 바꾸면 됩니다.

## 목표

총 플레이타임은 `15분`입니다. 15분까지 살아남으면 런 클리어입니다.  
오래 살아남으면서 적을 처치하고 경험치 보석을 먹어 레벨업합니다.  
레벨업할 때마다 3개의 선택지가 나오고, 그중 하나를 골라 성장합니다.

## 경험치통

첫 레벨업에 필요한 경험치는 `28`입니다.  
레벨업할 때마다 다음 경험치통은 아래 공식으로 커집니다.

```js
nextXp = Math.floor(nextXp * 1.24 + 12);
```

초반 필요 경험치는 대략 이렇습니다.

| 도달 레벨 | 필요한 경험치 |
| --- | ---: |
| 2 | 28 |
| 3 | 46 |
| 4 | 69 |
| 5 | 97 |
| 6 | 132 |
| 7 | 175 |
| 8 | 229 |
| 9 | 295 |
| 10 | 377 |

레벨업하면 체력이 최대 체력 안에서 `10` 회복됩니다.

## 기술과 업그레이드

현재 업그레이드는 총 `8개`입니다. 레벨업 시 이 중 무작위 3개가 선택지로 나옵니다.

| 이름 | 효과 | 추천 상황 |
| --- | --- | --- |
| 불꽃 씨앗 | 기본 투사체 피해량 `+7` | 초반 처치 속도가 부족할 때 |
| 빠른 심장 | 공격 간격 `12%` 감소 | 모든 공격 빌드에 무난함 |
| 갈래 줄기 | 투사체 수 `+1` | 단일 대상과 다수 처리 모두 좋음 |
| 달빛 발걸음 | 이동 속도 `+12%` | 포위망 탈출이 힘들 때 |
| 자석 향기 | 경험치 흡수 범위 `+45` | 레벨업 속도를 안정적으로 올릴 때 |
| 두꺼운 줄기 | 최대 체력 `+18`, 체력 `28` 회복 | 체력이 낮거나 생존이 불안할 때 |
| 회전 가시 | 주변 보호 가시 `+1` | 근접 적에게 둘러싸일 때 |
| 꽃가루 폭발 | 처치 시 폭발 확률 `+8%` | 적이 많이 몰릴수록 강함 |

## 공격 종류

현재 공격 계열은 크게 `3종`입니다.

| 공격 | 설명 |
| --- | --- |
| 기본 투사체 | 가장 가까운 적을 향해 자동 발사됩니다. 피해량, 공격 속도, 투사체 수 업그레이드의 영향을 받습니다. |
| 회전 가시 | 플레이어 주변을 도는 보호 무기입니다. 닿은 적에게 지속 피해를 줍니다. |
| 꽃가루 폭발 | 적 처치 시 확률로 주변 폭발이 발생합니다. 몰려오는 적을 연쇄 처리하기 좋습니다. |
| 필살기 | `Space`로 사용합니다. 화면 컷인 연출 후 전체 적에게 큰 피해를 줍니다. 쿨타임은 30초입니다. |

## 적 종류

적 체력과 속도는 생존 시간이 길어질수록 조금씩 증가합니다.

| 적 | 등장 시점 | 기본 특징 | 경험치 |
| --- | --- | --- | ---: |
| Husk | 처음부터 | 표준형 적 | 7 |
| Runner | 35초 이후 | 체력은 낮지만 빠름 | 6 |
| Brute | 75초 이후 | 느리지만 단단함 | 18 |
| Elite | 첫 25초 이후, 이후 40초마다 | 큰 체력과 높은 보상 | 70 |

## 난이도 상승

시간이 지날수록 적 생성 압박이 증가합니다.

| 항목 | 내용 |
| --- | --- |
| 적 생성 거리 | 플레이어 주변 바깥쪽 `560~820` 거리 |
| 기본 생성 간격 | 초반 약 `1.05초` |
| 최소 생성 간격 | `0.16초` |
| 생성 수 | 시간이 지날수록 증가 |
| 엘리트 | 첫 등장 25초, 이후 40초마다 |

## 추천 빌드

### 안정 성장 빌드

`자석 향기`를 1~2번 먼저 찍고, `갈래 줄기`, `빠른 심장`, `불꽃 씨앗`으로 공격력을 올립니다. 경험치를 덜 흘리기 때문에 레벨업 속도가 안정적입니다.

### 근접 생존 빌드

`회전 가시`와 `두꺼운 줄기`를 우선합니다. 적에게 둘러싸여도 버티기 쉽고, 컨트롤이 아직 익숙하지 않을 때 좋습니다.

### 폭발 연쇄 빌드

`꽃가루 폭발`, `갈래 줄기`, `빠른 심장`을 조합합니다. 적이 많아질수록 처치가 연쇄적으로 터져서 후반에 강합니다.

## 팁

- 초반에는 적을 죽이는 것보다 경험치 보석을 놓치지 않는 것이 중요합니다.
- `Runner`가 나오기 시작하는 35초 이후에는 이동 속도나 회전 가시가 체감됩니다.
- `Elite`는 경험치를 많이 주므로 무리하지 않는 선에서 잡는 것이 좋습니다.
- 체력이 낮을 때 `두꺼운 줄기`가 나오면 회복 수단으로 쓸 수 있습니다.
- 투사체 수를 늘리면 가까운 적 여러 명을 동시에 노릴 수 있어 안정성이 크게 오릅니다.
